# ✈️ Nine-Eleven

A fast-paced arcade experience inspired by *Flappy Bird*, built with **C# and WPF**.  
Take control of an aircraft, fight against gravity, and navigate through increasingly challenging obstacles.

## 🎮 About the Game

**Nine-Eleven** is a minimalistic yet engaging arcade game where precision and timing are everything.

Your mission is simple:
- Stay airborne  
- Avoid obstacles  
- Survive as long as possible  

Each obstacle you pass increases your score — but the difficulty never stops rising.


## ✨ Key Features

###  Smart Collision System
- The aircraft uses a **dynamic line-based hitbox**
- It rotates and tilts in real time вместе с движением самолета
- This makes collisions feel *accurate and responsive*

###  Smooth Animations
- Powered by `DispatcherTimer`
- Ensures stable frame updates and consistent gameplay

###  Atmospheric Visuals
- Parallax cloud background adds depth
- Stylized environment enhances immersion

##  Controls

| Key        | Action                     |
|------------|--------------------------|
| `Space`    | Jump / Gain altitude     |
| `Enter`    | Restart after game over  |
